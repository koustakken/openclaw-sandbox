# openclaw-sandbox

Monorepo with:
- `apps/web` — Vite + React + TypeScript
- `apps/api` — Express + TypeScript
- `packages/shared` — shared types/utilities

## Quick start

```bash
pnpm install
pnpm dev
```

## Structure

```text
apps/
  web/
  api/
packages/
  shared/
  tsconfig/
infra/
  docker/
```
